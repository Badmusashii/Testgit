        - Un dépôt Git (repositorie en anglais) est un endroit ou Git stocke les fichiers d'un 
        projet et enregistre les modifications au fil du temps, facilitant le suivi des versions et la 
        collaboration entre développeurs.


        - Un dépôt Git peut exister localement sur votre machine (dépôt local) ou être 
        hébergé sur une plateforme de gestion de code source telle que GitHub ou autre 
        (dépôt distant)
