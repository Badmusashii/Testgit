  • Une branche c'est quoi ?
        - La commande git add permet d’ajouter des ficheirs modifiés à l’index de Git, afin de 
        les préparer pour un commit ultérieur. Cela signifie que Git les surveillera pour les 
        inclure dans le prochain commit.
      
EX. git add pour ajouter des fichiers spécifiques
        Git add <nom_fichier>
        
        Ou tous les fichiers modifiés avec git add .
