La commande Git push

  • Acceuil
  • Un dépôt c'est quoi ?
  • Git clone
  • git commit
  • git add
  • git push
  • Une branche c'est quoi ?
        - La commande git push permet d’envoyer vos commits locaux vers un dépôt 
        distant (par exemple GitHub). Cela met à jour le dépôt distant avec vos dernières 
        modifications, permettant aux autres collaborateurs d’y acceder. Il suffit de faire git 
        push pour pousser vos commits vers le dépôt distant.
