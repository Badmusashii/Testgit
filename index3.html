  • Une branche c'est quoi ?
        - Cette commande permet de copier un dépôt Git existant (local ou distant) vers votre
        machine. Vous spécifiez l'Url du dépôt que vous souhaitez cloner, et Git crée une
        copie complète de tous les fichiers du projet sur votre machine.
