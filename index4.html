  • Une branche c'est quoi ?
        - Lorsque vous apportez des modifications à vos fichiers dans un dépôt Git, vous 
        utiliser la commende git –commit pour enrengistrer ces modifications dans 
        l’historique du dépôt. Vous associez un message descriptif à chaque commit pour 
        expliquer les changements effectués.

EX. git commit –m ''Ajout de nouvelles fonctionnalités''
